# drylabs.io

Company's website, [see it live](https://drylabs.io).

## Development

Make sure you have `ruby 2.5.1` installed,
use [`rbenv`](https://github.com/rbenv/rbenv) in case you don't.

1. `rbenv install`
2. `gem install bundler`

Then run:

1. `bundle install`
2. `bundle run jekyll s --safe`
3. Done!
